# 🌾 JeevanMitra AI

> **AI-Powered Smart Farming Assistant for Every Farmer**

JeevanMitra AI is a browser-based agricultural assistant that helps farmers make informed decisions using Artificial Intelligence. The platform provides crop recommendations, disease detection, yield prediction, live market insights, multilingual support, and voice interaction—all without requiring a backend server.

## ▶️ Run Locally

Clone the repository

```bash
git clone https://github.com/simha05/JeevanMitra-AI.git
```

Move into the project

```bash
cd JeevanMitra-AI
```

Start a local server

```bash
python -m http.server 8080
```

Open:

```
http://localhost:8080
```
